### 格式
```currency```支持两种格式:
- 币种名称 (```name```).
- 链编号@币种编号 (```chain_id@token_id```).
### 举例
以下表格中前两个币种的```currency```参数分别为:
- ```Ethereum``` 或 ```60@60```
- ```USDT-ERC20``` 或 ```60@0xdac17f958d2ee523a2206206994597c13d831ec7```
